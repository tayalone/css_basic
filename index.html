Where Css Belong to ?

  1. Inline ใน html tag
  2. Style Tag เขียน css script ใน header
  3. เขียนใน css file แล้ว import เข้ามา

Colors in Css

  1. Constant Color Code: เช่น 'red', 'green', 'blue' เป็นต้น
  2. Hexadecimal Color Code: เช่น '#E74C3C', '#48C9B0', '#E67E22' เป็นต้น
  3. RGB Color Code: เช่น 'rgb(130, 224, 170)', 'rgb(191, 201, 202)', 'rgb(169, 204, 227)' เป็นต้น
  4. RGBA Color Code (ค่าตัวสุดท้ายเป็น opicity): เช่น 'rgb(130, 224, 170,1)', 'rgb(191, 201, 202,0.5)', 'rgb(169, 204, 227,0.2)' เป็นต้น

CSS Selector

  1. Class Selector : เช่น "div", "h1", "p"
  2. Name Selector: ใช้ . ในการอ้าง name ใน html elemet เช่น ".name1", ".name2" เป็นต้น
  3. ID selector: ใช้ # ในการอ้าง ID ใน html element เช่น "#id1", "#id2"
  4. * : ใช้ css กับ ทุก element
  5. Decendant Selector : เช่น li a {} หมายถึง a ทุกตัวทั่อยู่ใน li
